# YDK core and bundle stubs

The `sdk` directory contains the `Python`, `C++` and `Go` code for YDK `core` package

```
sdk
├── _docsgen_common # Common documents for C++, Python, Go
├── cpp
│   ├── core # C++ core
│   ├── packages
│   ├── samples
│   └── tests
├── go
│   ├── core # Go core
│   └── packages
└── python
    ├── core # Python core
    └── packages

```
